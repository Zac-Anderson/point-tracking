## Getting Started ##
Follow the steps below to get setup.

### Prerequisites ###
- Java 11
- Gradle

```
brew install gradle
```

### Workspace Setup ###
```
cd ~/workspace
git clone git@github.com:Zac-Anderson/point-tracking.git
```

### Running Tests ###
```
cd ~/workspace/point-tracking
./scripts/test
```

### Starting the Server ###
```
cd ~/workspace/point-tracking
./scripts/dev-server
```
